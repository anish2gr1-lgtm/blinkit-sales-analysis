# Blinkit Sales Analysis (SQL + Power BI)

## Project Overview
This project analyzes Blinkit grocery sales data using SQL and Power BI to identify key revenue drivers, customer trends, and outlet performance.

## Objectives
- Calculate key KPIs such as Total Sales, Average Sales, and Customer Rating
- Identify top-performing products and outlets
- Build interactive dashboard for business monitoring
- Automate reporting workflow

## Tools Used
- SQL (MySQL / PostgreSQL)
- Power BI
- Power Query
- Excel / CSV

## Project Workflow
1. Imported raw CSV data into SQL database
2. Performed data cleaning and transformation using SQL
3. Calculated KPIs using SQL aggregations
4. Created analytical views for reporting
5. Connected Power BI to SQL dataset
6. Built interactive dashboard

## Key Insights
- Identified top-performing outlet types
- Found highest revenue-generating product categories
- Analyzed sales performance by location and outlet size
